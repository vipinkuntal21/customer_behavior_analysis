# customer_behavior_analysis
Overview
This project focuses on analyzing customer behavior using data analytics techniques to generate meaningful business insights. The workflow includes loading and processing datasets in Python, performing Exploratory Data Analysis (EDA), cleaning and transforming data, running SQL queries on a MySQL server, and building an interactive Power BI dashboard for visualization and reporting.

The project helps understand customer purchasing patterns, sales trends, and key performance indicators to support data-driven decision-making.

Dataset

The dataset contains customer transaction and sales-related information such as:

Customer demographics
Product details
Purchase history
Sales and revenue data
Customer engagement metrics

How to Run
Clone the repository.

Install required Python libraries:

pip install pandas numpy matplotlib seaborn mysql-connector-python
Open the Jupyter Notebook and run the analysis scripts.
Connect MySQL database for SQL queries.
Open the Power BI dashboard file to explore insights.
